# Web Radar Collector Chrome Extension

一个本地优先的 Chrome MV3 插件，用来从任意网页手动采集选中文本、图片和视频暂停帧，并导出为 Markdown 表格和本地素材文件夹。它适合做资料整理、竞品观察、用户反馈摘录和轻量研究采样。

## 特点

- 手动选择采集内容，不自动批量抓取网页。
- 支持选中文本、图片截图和视频暂停帧。
- 导出 `任务名.md` 和 `assets/`，方便分享、归档和继续整理。
- 本地运行、本地下载，不需要服务器。

## 安装

1. 打开 Chrome `chrome://extensions/`。
2. 打开右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择解压后的插件目录，也就是包含 `manifest.json` 的那个文件夹。

## 使用

1. 点击浏览器工具栏里的扩展图标，新建任务并输入任务名。
2. 选中一段文字，松开鼠标后点击旁边的「采集」按钮。
3. hover 图片或视频，会出现「采集」按钮；鼠标停留在媒体 120% 范围内时按钮不会消失。
4. 点击 popup 里的「结束并导出」会下载一个本地任务文件夹。
5. 20 分钟没有新增采集时，插件会自动结束并导出。

图文版说明见：[USAGE.md](./USAGE.md)。

## 导出内容

Chrome 扩展不能静默写入任意磁盘目录，所以文件会下载到浏览器默认下载目录里的子文件夹。

文件夹名格式为：

```text
yyyy-mm-dd hh：mm：ss 任务名
```

Windows 路径不能使用英文冒号，所以实际文件夹名使用全角冒号 `：`。文件夹内包含：

- `任务名.md`：本次任务的 Markdown 表格记录
- `assets/`：采集到的图片和视频暂停帧

Markdown 表格包含「编号」「页面」「采集内容」三列；页面列是超链接，采集内容列可以包含文字和截图。

## 限制

- 图片和视频优先保存页面可见区域截图，避免跨域或防盗链导致媒体文件下载失败。
- 媒体截图只能截取当前视口内可见的部分；被遮挡或滚出屏幕的区域不会出现在截图里。
- 采集大量高清截图会占用 Chrome 本地扩展存储，建议按任务分批结束导出。

## 使用边界

请只在你有权访问和整理的网页内容上使用本工具，并遵守目标网站的服务条款和相关法律法规。本项目不提供登录绕过、反爬绕过或自动化批量抓取能力。
